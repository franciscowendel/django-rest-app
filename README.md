# django-rest-project

A Django project with a lot of DRF (Django Rest Framework) on it.

Como executar o projeto:

1. Caso tenha o pipenv instalado:
```console
pip install pipenv
pipenv sync -d
```

2. Caso tenha o pipenv instalado:
```console
pipenv sync -d
```

3. Crie o arquivo .env e dentro deste crie as seguintes vari√°veis de ambiente:
- DEBUG=True
- ALLOWED_HOSTS=localhost,127.0.0.1
- SECRET_KEY=minhachave
